<h2>궁금한점</h2>

<h3>1. Date() 함수랑 new의 역할이 뭔지 모르겠음.</h3>

> Date()는 생성자 함수로 어떻게 호출하느냐에 따라 동작이 달라짐

| 구분       | 설명                                                                                                                |
| ---------- | ------------------------------------------------------------------------------------------------------------------- |
| new Date() | Date 객체를 생성 / 여기서 new는 새로운 객체를 만들어 달라는 의미로 Date 객체 안에는 날짜/ 시간 관련 기능이 들어있음 |
| Date()     | 그냥 함수 호출로, 객체를 안만들고, 현재 날짜/시간을 문자열로 반환                                                   |

> 쉽게 이해하기

| 구분                 | 비유            | 코드                       |
| -------------------- | --------------- | -------------------------- |
| 생성자 (Constructor) | 붕어빵 틀       | Date                       |
| new                  | 굽는 행위       | new                        |
| 인스턴스 (Instance)  | 만들어진 붕어빵 | const myDate = new Date(); |

<h3>2. 또 헷갈리네, textContent / innerText / innerHtml 다시 정리</h3>

| 속성태그    | 태그 포함 | 숨겨진 텍스트 | 특징             |
| ----------- | --------- | ------------- | ---------------- |
| textContent | ❌        | ✅ 포함       | 가장 빠르고 정확 |
| innerText   | ❌        | ❌ 제외       | CSS 영향 받음    |
| innerHTML   | ✅        | ❌            | HTML 자체        |

<h3>3. 변수 만들때 const랑 let 사용하는걸로 아는데, 어떻게 차이 있는지 알려줘</h3>

| 구분        | const      | let     |
| ----------- | ---------- | ------- |
| 재할당      | ❌ 불가    | ✅ 가능 |
| 선언 시 값  | 필요 ✅    | ❌      |
| 블록 스코프 | ✅         | ✅      |
| 추천 사용도 | ⭐⭐⭐⭐⭐ | ⭐⭐⭐  |

<h3>4. let hr = date.getHours; 하니까 메소드 자체가 콘솔에 찍히는데, 어떻게 하면 텍스트만 추출하지?</h3>

> 메소드를 호출하려면 ( )를 써야한다. 안쓰면 함수 자체가 콘솔에 찍힘

| 구분            | 설명             |
| --------------- | ---------------- |
| date.getHours   | 함수 자체 (참조) |
| date.getHours() | 함수 실행 결과   |
